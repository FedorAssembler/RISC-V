# RISC-V
Это список моих мини-проектов по CS и CE реализованные через программирование под архитектуру RISC-V на ассемблере RV64i/RV32i.

# Идея

Мне понравилась идея написать список своих программ на ассемблере и выложить его на GitHub. 
Список программ будет пополняться новыми файлами по мере моего обучения.
Большинство проектов будет реализовано как программы в одном файле с комментариями к коду. 
Все программы написаны без эксплуатации больших языковых моделей.
